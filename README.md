# SQL Analytics & Window Functions Practice — PostgreSQL

A comprehensive collection of SQL queries built on a 
retail dataset (customers, products, sales) covering 
foundational to advanced PostgreSQL concepts.

## Database Schema
- **customers** - customer details and membership status
- **products** - product catalog with categories and pricing
- **sales** - transaction records linking customers and products

## Concepts Covered
- Basic SELECT, WHERE, GROUP BY, HAVING, ORDER BY
- JOINS — INNER, LEFT, RIGHT, CROSS
- Aggregate functions — SUM, COUNT, AVG, MAX, MIN
- Subqueries — scalar, derived tables
- CTEs (Common Table Expressions) using WITH
- Window functions:
  - RANK(), DENSE_RANK()
  - NTILE()
  - PERCENT_RANK()
  - LAG(), LEAD()
  - SUM() OVER(), AVG() OVER()
- Date functions — date_trunc(), extract()

## Tools Used
- PostgreSQL
- DBeaver
